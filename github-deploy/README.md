# 始于技术，终于组织 · AI 时代的组织转型与 HR 的破局跃迁

面向 HRD / HR 群体的分享演示（科技酷炫风 · 全屏 Keynote）。

## 在线访问

部署到 GitHub Pages 后，访问地址为：

```
https://<你的GitHub用户名>.github.io/<仓库名>/
```

例如：`https://wisdomnexus.github.io/ai-hr-keynote-20260627/`

## 操作方式

- 翻页：键盘 ← / → 、空格、鼠标滚轮、点击屏幕、手机触屏滑动
- 进度：顶部进度条、右上角页码、底部圆点
- 深链：地址栏 `#页码` 可直达指定页

## 文件说明

| 文件 / 目录    | 说明                                  |
| -------------- | ------------------------------------- |
| `index.html`   | 演示主文件（单文件，已内联所有样式与脚本） |
| `img/`         | 图片资源（头像、动作演示图、二维码等）   |
| `.nojekyll`    | 关闭 GitHub Pages 的 Jekyll 处理（必需） |
| `README.md`    | 本说明文件                            |

## 部署要点

1. 仓库必须为 **Public（公开）**，免费版 GitHub Pages 才能使用。
2. `index.html` 与 `img/`、`.nojekyll` 必须在**仓库根目录**（不要套一层子文件夹）。
3. 在仓库 Settings → Pages → Build and deployment 中，选择 **Deploy from a branch**，分支选 `main`，目录选 `/ (root)`。
